# Refeições App

Este é um aplicativo de gerenciamento de refeições desenvolvido em React Native + Expo. Ele permite que os usuários criem, visualizem e gerenciem suas refeições diárias, ajudando-os a manter uma alimentação saudável e organizada.

## Funcionalidades

- **Adicionar Refeições**: Os usuários podem adicionar novas refeições, incluindo detalhes como nome, descrição e horário.
- **Visualizar Refeições**: As refeições adicionadas são exibidas em uma lista, permitindo que os usuários vejam todas as suas refeições diárias de forma organizada.
- **Gerenciar Refeições**: Os usuários podem editar ou excluir refeições existentes, facilitando a manutenção de um registro atualizado de suas refeições.
- **Estatísticas de Refeições**: O aplicativo pode fornecer estatísticas sobre as refeições, como o número total de refeições, e intervalos entre as refeições.

## Tecnologias Utilizadas
- **React Native**: Para o desenvolvimento da interface do usuário e lógica do aplicativo.
- **Expo**: Para facilitar o desenvolvimento e a implantação do aplicativo em dispositivos móveis.

## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/NakeleKantoo/RefeicoesApp.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd RefeicoesApp
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o aplicativo:
   ```bash
   npm run start
   ```
